★ 종목 분석 마스터 플로우

김도아 교육이사의 종목 분석 → 매수 → 매도까지의 전체 흐름입니다. 이 순서대로 따라하면 실전 매매가 가능합니다.

8단계 마스터 플로우 — 이 순서를 외우세요
1. 큰 흐름→ 2. 종목 선정→ 3. 기업분석→ 4. 차트분석→ 5. 수급확인→ 6. 매수→ 7. 목표가→ 8. 매도

Step 1. 큰 흐름 확인 (탑다운)

Top-Down // 해외 → 국내 → 섹터 → 종목 순서로 좁혀간다 해외: 미국 나스닥/S&P, 금리, 유가, 환율 방향 확인 국내: 코스피/코스닥 추세, 외인 수급 방향 섹터: 어떤 섹터가 3일+ 연속 강세인가? 키워드: 주말사이 핵심 이슈 정리 // 실전 인용 (장건 회장 3/9) "주말사이 키워드: 유가상승(감산), 스테그플레이션, 블랙록 환매 첫 제한, 이란 모즈타파(차남), 신규상장 엑스비스"
"해외-국내 흐름따라 종목선정, 종목이 괜찮으면 자세한 분석, 우량하고 전망있으면, 배당율도 있다"
— 김도아 교육이사 (정모 강의)

Step 2. 종목 선정

종목은 이렇게 찾는다
  1. 장마감 상승률 TOP 20 확인 — 어떤 섹터가 올랐나?
  2. 3일 연속 반복되는 종목/섹터 체크 — "테마는 순환합니다. 계속 등장하는 종목들 이름 외우기" (장건 회장 3/18)
  3. 섹터 내 1위/2위 기업만 선별 — "독점적 지위, 예를 들면 HPSP. 영업이익률 50% 넘어요" (장건 회장 3/15)
  4. 산업 트렌드에 맞는가? — "트렌드를 따라가면서 지속성장 이어가는가?"
"한미반도체, ISC, 대덕전자, 이수페타시스, 이오테크닉스, HPSP, 테크윙, SK하이닉스, 리노공업 중에서 한두놈은 조정구간에서 붙들고 있으면 좋겠다. 추격매수 금지, 조정구간 분할 매수~~ 10%정도는 꼬옥!!"
— 김영창 (3/10, 전쟁 폭락 직후)
장건식 좋은 매수 후보 4요소 (CJ 사례)

장건 회장이 CJ를 관심 종목으로 꼽았을 때의 판단 기준 (2025-10-01):

  1. 차트: 주봉 쌍바닥 패턴 — "주봉 쌍바닥 패턴이 눈에 들고"
  2. 재료: 구체적 호재 — 한한령 수혜주
  3. 밸류: 저평가 — 저 PBR
  4. 수급: 기관/외인 유입 (동반 확인)

4요소가 겹치면 강력한 매수 후보. 차트만, 재료만, 밸류만으로는 부족.

Step 3. 기업 기본분석 (핵심!)

Fundamental Analysis // 도아 회장의 기업분석 체크 포인트 실적 3년 동기대비 • 1분기 대비 1분기로 비교 (분기별) • "고만고만 하면 안돼. 껑충뛰는 숫자가 나오면 눈여겨봐" • "적자였다 확 돌아서면 눈여겨 봐야해" • 영업이익이 곱배기가 되면 좋지! PER (주가수익비율) • 업종 평균보다 낮으면 저평가 • "PER 떨어지면 돈 잘 번다는 말" • 삼성전자 최근 PER 하락 = 실적 개선 신호 PBR (주가순자산비율) • 기준은 1 — "주식의 시가총액, 빚을 다 주고 받을게 있냐?" • PBR 1 이상이면 줄 게 더 많다 • PBR 3 이상이면 유보율, 배당도 확인 ROE (자기자본이익률) • 내 돈(자본)으로 얼마나 벌었나 = 수익성 핵심 지표 • ROE 15%+ = 우량 기업 (워런 버핏 기준) • ROE가 매년 올라가면 = 기업이 점점 잘 버는 중 부채비율 • "현금에 비해 빚이 30%면 재무구조 탄탄하다" 내부자 거래 • 내부자 매수 있으면 3~6개월 후 반응 • 지분변동, 배당, 유보율 함께 체크
도아 회장의 적정주가 산출법

"3년치 또는 4년치 영업이익 합으로 평균내고 주가도 3년치의 고점 저점 딱 중간의 주가. 평균 영업이익에서 올, 내년 추정치가 얼마정도 증가하는지 확인" (3/17)

  1. 최근 3~4년 영업이익 평균 계산
  2. 같은 기간 주가 고점+저점의 중간값 = 기준 주가
  3. 올해/내년 영업이익 추정치의 증가율 확인
  4. 기준 주가 × 증가율 = 적정주가
  5. 현재 주가 < 적정주가 = 저평가 (매수 후보)

Step 4. 차트 기술적 분석

Technical Analysis Flow // 큰 것부터 본다 — 월봉 → 주봉 → 일봉 1) 방향 (추세) • 상승추세? 하락추세? 횡보(박스권)? • "차트는 방향을 본다. 추세가 핵심" 2) 이동평균선 (이평선) • 정배열 = 정상 (5일 > 20일 > 60일 > 120일) • 역배열 = 약세 (반대 순서) • "이평선이 모아져 있으면 사야지" • "60선은 실적선. 기관들이 좋아해" • 장기이평 뚫을때는 머리를 세번 맞는다 (3번 저항) 3) 박스권 & 돌파 • 고점-저점 사이 = 박스권 • "돌파를 하면 위에서 놀아야. 고점 돌파 = 아주 좋은 신호" • 횡보가 길수록 폭발력이 강하다 4) 기준봉 & 눌림목 • 돌파할 때 강한 양봉 = 기준봉 • "그 다음 봉의 반을 그어. 그게 눌림목의 기준" • 눌림목까지 내려오는지 안 내려오는지 확인 • 이평선 + 매물대 지지 라인과 겹치면 최고의 매수 자리 5) 이격도 & 회귀본능 • 중소형주 200% 오르면 회귀본능 • 대형주 150% 오르면 회귀본능 • "이격도가 커지면 정상의 자리로" • 150~200% 상승했으면 위험 → 분할매도 • "이쁜 차트는 계단식, 차근차근"

Step 4-2. 역배열 돌파 = 최고의 매수 기회

이 조합이 나오면 강력한 매수 후보

실적이 개선되는 기업 + 저점/장기 횡보 구간 + 역배열 끝내고 + 장기이평 돌파
= 단순 반등이 아니라 구조 변화의 신호

좋은 돌파 vs 나쁜 돌파 좋은 돌파: • 장기 횡보 후 상단 돌파 • 거래량 증가 동반 • 돌파 후 상단 위 안착 (고점 위에서 놈) • 실적/산업 모멘텀 동반 • 이평선이 수렴하며 정배열 초입 나쁜 돌파 (속지 마세요): • 실적 뒷받침 없음 • 하루짜리 뉴스 재료로 급등 • 돌파 후 다시 박스 안으로 복귀 • 거래량 없이 슬금슬금 올라간 것
종목은 미친듯이 파라

"종목을 알고 싶으면 미친듯이 파라. 그러면 박스권 매매를 할 수 있다"
얕게 50종목보다 깊게 5종목이 실전적입니다. 깊게 파야 박스권/이벤트/수급 변화에 대응할 수 있어요.

Step 5. 수급 확인

수급 체크 포인트

• 쌍끌이 수급 = 가장 강한 매수 신호 (아래 상세 참조)
• 투신 리밸런싱: 월말 매도 → 월초 재매수 (단기 큰 수익 내야 하는 주체)
• 외인 선물 포지션: "오늘의 관전 포인트는 외인의 선물 매스 포지션" (김영창 3/9)
• 코스닥150 vs 코스닥: 150이 강하면 산업형 상승 = 건강

수급 조합외국인기관개인신호 강도의미
쌍끌이 매수 매수 매수 매도 최강 프로(외인+기관)가 사고 개인이 파는 구간 = 상승 확률 가장 높음
외인 주도 매수 매도 매도 강 외인 단독 매수 = 글로벌 자금 유입, 방향성 있는 매수
기관 주도 매도 매수 매도 중강 기관 실적 기반 매수 = 펀더멘털 양호 신호
개인 주도 매도 매도 매수 위험 개인만 사고 프로가 파는 구간 = 고점 주의, 역지표
쌍끌이 매도 매도 매도 매수 최악 프로가 다 파는데 개인만 사는 구간 = 하락 확률 가장 높음
쌍끌이 실전 활용법

• 쌍끌이 매수 + 거래량 증가 = 최고의 매수 신호
• 쌍끌이 매수가 3일+ 지속 = 추세적 상승 시작 가능성
• 쌍끌이 매도 + 거래량 증가 = 즉시 비중 축소 or 손절 검토
• 개인만 매수 = "나만 사고 있다" 느낌이면 위험 신호
• 확인 방법: HTS > 투자자별 매매동향 (일별/주별)

Step 6. 매수 실행

Buy Execution // 매수는 이 조건이 겹칠 때만 최적 매수 자리: • 이평선이 수렴(모여있는) 구간 • 장기이평 근처에 주가가 있을 때 • 정배열 만들어지는 초입 • 기준봉 절반 눌림목에서 분할매수 실행: 1차: -5% 구간 (비중 30%) 2차: -10% 구간 (비중 30%) 3차: 반등 확인 후 (나머지 40%) 총 비중: 포트의 10% 이내 // 실전 인용 "추격매수 금지, 조정구간 분할 매수~~ 10%정도는 꼬옥!!" — 김영창 (3/10) "저가를 얻으려면 시간투자를 해야해" — 도아 교육이사 (정모)
실전 매수 후보 7가지 조건 — 이게 다 겹치면 강한 후보
  • 산업 흐름이 좋다 (트렌드와 맞물림)
  • 실적이 개선되고 있다 (적자→흑자, 영업이익 껑충)
  • 장기 횡보가 끝나는 구간이다
  • 역배열이 해소되고 있다
  • 장기이평을 돌파했다 (머리 3번 맞고)
  • 거래량이 유입되고 있다
  • 돌파 후 안착했다 (고점 위에서 놀고 있다)
"감을 믿지 말고 계기판을 봐라. 거래량, 수급, 이평선, 실적 — 이게 계기판이다."
— 정모 강의 핵심 원칙

Step 7. 목표가 설정

Target Price // 매수하면서 동시에 목표가를 정한다 방법 1: 박스권 배수 • 박스권 평균에서 몇 배 갔는지 계산 • 예: 255,000원 기준 × 2 = 목표가 510,000원 • "30% 성장이면 그 정도까지는 간다고 보는 거지" 방법 2: 이격도 기준 • 중소형: 200% 도달 시 = 과열 → 분할매도 • 대형: 150% 도달 시 = 과열 → 분할매도 방법 3: 적정주가 대비 • 3~4년 평균 영업이익 기반 적정주가 • 적정주가 도달 or 초과 시 = 매도 검토 // 실전 인용 "삼천당 100만원 간다" — 도아 교육이사 (3/20) → 실제 100만원 도달 (3/24) → 300%+ 수익

Step 8. 매도 실행

매도 3원칙

1) 목표가 매도: 적정주가 도달 시
2) 추세 매도: "추세가 꺽이지 않을때는 전량 매도가 아닌 원금 찾고 추세보면서 매도" (도아 3/25)
3) 이격 매도: 150~200% 상승 시 분할매도

"흔들리는거에는 겁먹지 말아야 해요. 차트는 살아있기에... 지켜보는 것이고"
— 김도아 교육이사 (3/25, 삼천당 급락 시)

★ 실전 사례: 솔브레인 분석

4/11 정모에서 도아 교육이사가 실제로 시연한 종목 분석 흐름입니다. 마스터 플로우 8단계가 실전에서 어떻게 적용되는지 보여줍니다.

Step 1~2: 산업 확인 → 종목 선정

왜 솔브레인인가?

• 반도체 소재 기업 (금속배선/CMP 공정 핵심)
• 반도체 섹터가 전쟁 중에도 목표주가 상향된 유일한 섹터
• 장건 회장이 매일 언급하는 반도체 소부장 1위/2위 기업군

Step 3: 기업 기본분석

Part 3. 차트 분석 실전

차트를 읽는 것은 시장의 심리를 읽는 것입니다. 주봉 중심의 실전 분석법을 정리합니다.

3-1. 주봉 vs 일봉 vs 월봉

주봉 (Weekly)

중기 추세 판단 — 스윙매매의 핵심 차트. 김도아 이사가 주로 사용하는 타임프레임. 노이즈가 적고 추세가 명확.

일봉 (Daily)

단기 진입 타이밍 — 주봉에서 방향을 잡은 뒤, 일봉으로 세부 진입 포인트를 찾는다.

월봉 (Monthly)

대세 판단 — "월봉 쌍바닥에 5년짜리 컵핸들" - 김영창의 실제 코스닥 분석에서 나온 표현.

3-2. 파동 분석 기초 (김도아식)

Wave Pattern // 5파동 구조 — 상승 추세의 기본 패턴 1파동: 첫 상승 → 시장의 관심 시작 | 2파동: 조정 → ★ 매수 타이밍 ★ (눌림목) | 3파동: 본격 상승 → 수익 구간 (가장 길고 강한 파동) | 4파동: 재조정 → 추가 매수 or 관망 | 5파동: 마지막 상승 → 매도 준비 // 5파동 끝나면 A-B-C 조정파동이 온다 A파: 첫 하락 → 많은 사람이 "눌림목이겠지" 착각 B파: 반등 → 함정! 데드캣 바운스 (가짜 반등) C파: 본격 하락 → 추세 전환 확정. 손절 or 관망

3-3. 눌림목 매수법

눌림목이란?

상승 추세에서 일시적으로 조정받는 구간. 2파동 조정 시 나타나며, 가장 안전한 매수 타이밍입니다.

눌림목 판단 실전 3단계 Step 1. 주봉 차트 열기 → 최근 상승 후 조정 구간 확인 Step 2. 거래량 바 확인: • 거래량 줄면서 하락 = 매도 세력 소진 = 매수 OK • 거래량 늘면서 하락 = 추세 전환 가능 = 매수 금지 Step 3. 20일 이동평균선 지지 여부 확인 • 20일선 위에서 반등 → 강한 눌림목 • 20일선 이탈 → 60일선까지 기다리거나 관망
실전 팁: 캔들 패턴으로 반등 확인

눌림목 바닥에서 양봉(빨간 캔들)이 나오면 반등 시작 신호. 특히 아래꼬리가 긴 양봉은 매수세 유입을 뜻합니다. 이때가 진입 타이밍.

매수 금지 신호

거래량이 늘면서 하락하면 추세 전환 가능성이 높습니다. 이때는 절대 매수하지 마세요.

Part 4. 수급 분석법

4-1. 3대 주체 읽기

외국인

시장 방향의 선행지표. 외인 매수 전환 = 상승 신호. 가장 중요한 수급 주체.

기관

실적 기반 매매. 기관 매수 = 펀더멘털 양호. 안정적이고 지속적인 매매 패턴.

개인

역지표로 활용. 개인 매수 집중 = 고점 주의. 개인이 몰리면 조심해야 합니다.

Part 5. 매수 타이밍 완전 정복

언제, 얼마나, 어떻게 사는지가 수익의 80%를 결정합니다.

5-1. 매수 5원칙

단톡방에서 종합한 매수의 핵심 원칙입니다.

  1. 섹터 확인 먼저, 종목은 나중에 — 상승 섹터를 찾은 후 그 안에서 종목 선별
  2. 1위/2위 기업만 — 장건 회장: "반도체 공정별 1위/2위 기업만 선별"
  3. 조정구간 분할매수 — 추격매수 절대 금지. 눌림목을 기다린다
  4. 비중 10% 이하 — H.S: "개별종목 전체 비중 10% 이하"
  5. 동시호가 관찰 — 장건 회장: "장시작 동시호가에 들어오는 것이 보여요"

5-2. 분할매수 실전법 (김영창 스타일)

분할매수 전략 // 3단계 분할매수 — 총 비중은 포트의 10% 이내 1차 매수: 조정 시작 후 -5% 구간 → 전체 비중의 30% 2차 매수: 추가 하락 -10% 구간 → 전체 비중의 30% 3차 매수: 바닥 확인 후 반등 시 → 나머지 40% // 핵심: 한 번에 다 사지 않는다. 3번에 나눠 산다. // 총 비중: 해당 종목이 전체 포트의 10% 이내

5-3. 종목 선별 실전 프로세스

종목 찾기 5단계 (장건 회장 + 김도아 이사 종합)
  1. 장마감 후 상승률 TOP 20 확인 — 어떤 섹터가 올랐나?
  2. 3일 연속 같은 섹터 상승 확인 — 테마 형성 중인가?
  3. 해당 섹터 내 시가총액 1위, 2위 기업 선별
  4. 선별 종목의 영업이익 증가율 확인 (3~4년 평균 대비 30%+)
  5. 주봉 차트에서 눌림목(2파동 조정) 확인 후 분할매수 진입

5-4. 적정주가 계산 실전 (김도아식)

적정주가 계산 예시 // SK하이닉스 예시 (가상 수치) 1단계. 최근 4년 평균 영업이익 2023: 1.5조 / 2024: 7.7조 / 2025: 23.5조 / 2026E: 30조 평균 = 15.7조 2단계. 올해 컨센서스 = 30조 3단계. 증가율 = (30 - 15.7) / 15.7 × 100 = +91% 4단계. 증가율 91% > 30% → 매수 후보 OK 현재 주가가 이 실적을 반영했는지 PER로 재확인 5단계. 동종업계 평균 PER × 올해 EPS = 적정주가 적정주가 대비 현재 주가가 낮으면 저평가 = 매수
PER 빠르게 확인하는 법

네이버 증권 → 종목 검색 → 투자정보 탭 → PER, 추정 EPS 확인. 업종 평균 PER보다 낮으면 저평가 가능성.

5-5. 매수 금지 상황

이런 상황에서는 절대 매수하지 마세요
  • 이미 10%+ 상승한 종목 추격
  • 상한가 다음날 매수 (상따)
  • 뉴스만 보고 묻지마 매수
  • 신용/미수 사용
  • 현금비중 0%인 상태에서 추가 매수

Part 6. 매도 타이밍 & 손절

매수보다 매도가 어렵습니다. 사전에 매도 기준을 세우는 것이 핵심입니다.

6-1. 매도 3가지 유형

목표가 매도

김도아식 적정주가 계산으로 산출한 목표가에 도달 시 매도. 감정 배제, 숫자로 판단.

실적 매도

실적 발표일 갭상승 → 당일 or 익일 매도. 장건 회장의 핵심 스윙 전략.

손절 매도

사전 설정한 손절선 도달 시 기계적 매도. 망설이면 손실이 커진다.

6-2. 실적 시즌 매매법 (장건 회장 핵심 전략)

  1. 실적 발표 2~3주 전: 좋은 실적 예상 종목 리스트업
  2. 조정구간에서 분할매수: 실적 발표 전 눌림목에서 진입
  3. 실적 발표일: 갭상승 확인
  4. 갭상승 당일 or 익일: 매도 실행
  5. 핵심 원리: "실적은 주가에 선반영된다" — 발표 후 오히려 하락할 수 있음

6-3. 목표 수익률 설정법

목표가 설정 실전 // 매수할 때 반드시 3개 가격을 미리 정한다 손절가: 매수가 × 0.93 = -7% // 이 가격 오면 무조건 판다 1차 목표: 매수가 × 1.10 = +10% // 절반 매도 (본전 확보) 2차 목표: 매수가 × 1.20 = +20% // 나머지 매도 (수익 확정) // 예시: 10만원에 100주 매수 시 손절가: 93,000원 → 100주 전량 매도 (손실 -70만원) 1차: 110,000원 → 50주 매도 (수익 +50만원, 나머지 50주 본전) 2차: 120,000원 → 50주 매도 (추가 수익 +50만원)

6-4. 손절 원칙

손절은 실패가 아니라 자산 보호
  • 매수 시점에 손절선 사전 설정 (권장 -7% ~ -10%)
  • "다시 오르겠지" 희망회로 작동 시 → 즉시 손절
  • 전쟁 같은 외부 충격은 예외 (패닉셀 금지, 대응 매매)
  • 손절 후 같은 종목 복수매매 금지

Part 12. 캔들 패턴 실전

복잡한 이론 다 필요 없고, 이 6가지만 알면 됩니다.

12-1. 반등(매수) 신호 캔들

망치형 (Hammer)

아래꼬리가 몸통의 2배 이상 긴 양봉.
→ "밑에서 강하게 사올렸다" = 바닥 반등 신호
하락 추세 끝에서 나오면 매수 검토

상승 장악형

전일 음봉을 완전히 감싸는 큰 양봉.
→ "매수세가 매도세를 압도" = 추세 전환 신호
거래량 증가 동반 시 신뢰도 높음

샛별형 (Morning Star)

음봉 → 십자형(작은봉) → 양봉 3일 패턴.
→ "하락 멈추고 반등 시작" = 강력한 매수 신호
주봉에서 나오면 매우 신뢰도 높음

12-2. 하락(매도) 신호 캔들

교수형 (Hanging Man)

상승 추세 끝에서 나오는 아래꼬리 긴 봉.
→ "고점에서 매도 압력 시작" = 하락 전환 경고
다음날 음봉이면 매도

하락 장악형

전일 양봉을 완전히 감싸는 큰 음봉.
→ "매도세가 매수세를 압도" = 하락 전환 신호
고점 부근에서 나오면 즉시 매도 검토

석별형 (Evening Star)

양봉 → 십자형(작은봉) → 음봉 3일 패턴.
→ "상승 멈추고 하락 시작" = 강력한 매도 신호
목표가 부근에서 나오면 확실한 매도

12-3. 추세 전환 신호 캔들

도지 (십자형)

시가 = 종가, 위아래 꼬리만 있는 형태.
→ "매수세와 매도세가 팽팽" = 방향 전환 임박 신호
고점에서 도지 = 하락 전환 / 저점에서 도지 = 반등 가능

캔들 패턴 활용 팁

단독으로 판단하지 말 것. 반드시 거래량 + 이동평균선 + 위치(고점/저점)와 함께 봐야 합니다.
• 저점에서 망치형 + 거래량 증가 = 신뢰도 높음
• 고점에서 교수형 + 거래량 증가 = 신뢰도 높음
• 고점/저점에서 도지 = 다음 봉 방향 주목
• 횡보구간에서의 캔들 패턴 = 신뢰도 낮음 (무시)

Part 17. 거래량 심화 분석

"가격은 속일 수 있지만, 거래량은 속일 수 없다." 거래량은 시장 참여자들의 진짜 의지를 보여줍니다.

17-1. 거래량 + 주가 조합 해석

주가거래량의미행동
상승 증가 강한 매수세 유입 = 건강한 상승 보유 유지 or 매수
상승 감소 매수세 약화 = 상승 피로, 고점 주의 비중 축소 검토
하락 증가 강한 매도세 = 추세 전환, 투매 매수 금지, 손절 검토
하락 감소 매도세 소진 = 바닥 근접 가능 눌림목 매수 준비

17-2. 거래량 핵심 패턴 3가지

거래량 돌파 (Volume Breakout)

평소 거래량의 3배 이상이 터지면서 양봉 → 강력한 매수 신호.
특히 횡보 후 돌파 시 신뢰도 최고.
실전: 당일 매수 or 익일 시초가 매수

거래량 클라이맥스 (Volume Climax)

역대급 거래량 + 장대 음봉 = 패닉셀 절정.
이 다음 날부터 반등 가능성 높음.
실전: 당일 매수 금지, 익일~2일 후 반등 확인 후 분할매수

거래량 수축 (Volume Dry-up)

거래량이 극도로 줄어든 상태 = 관심 소멸 or 에너지 축적.
이후 방향이 결정되면 크게 움직임.
실전: 터지는 방향 확인 후 따라가기

17-3. 거래량으로 세력 움직임 읽기

Smart Money Tracking // 세력(기관/외인)의 매집 패턴 매집 단계: 주가 횡보 + 거래량 서서히 증가 → 누군가 조용히 사고 있다 = 관심 종목 등록 돌파 단계: 갑자기 거래량 3배+ 폭발 + 양봉 → 매집 끝나고 올리기 시작 = 매수 타이밍 분배 단계: 고점에서 거래량 증가 + 주가 횡보/소폭 하락 → 세력이 물량 넘기는 중 = 매도 준비 하락 단계: 거래량 감소 + 지속 하락 → 세력 빠져나감 = 매수 금지

Part 19. 리스크 관리 & 투자 심리

기술적 분석보다 중요한 게 리스크 관리와 심리 통제입니다. 90%의 개인 투자자가 지는 이유는 실력이 아니라 심리입니다.

19-1. 리스크/리워드 비율

Risk/Reward Ratio // 매수 전 반드시 계산하는 것 리스크 = 매수가 - 손절가 = 잃을 수 있는 금액 리워드 = 목표가 - 매수가 = 벌 수 있는 금액 R/R 비율 = 리워드 / 리스크 // 예시: SK하이닉스 매수가: 935,000원 손절가: 870,000원 (리스크 = 65,000원) 목표가: 1,060,000원 (리워드 = 125,000원) R/R = 125,000 / 65,000 = 1.92 // 기준 R/R 2:1 이상 = 매수 가치 있음 (2만원 벌 때 1만원 잃을 각오) R/R 1.5:1 = 보통 (확신 있을 때만) R/R 1:1 이하 = 매수 금지 (리스크 대비 보상 부족)

19-2. 포지션 사이징 (얼마나 살 것인가)

2% 룰 — 한 번의 매매에서 총 자산의 2% 이상 잃지 않는다
  1. 총 투자금: 5,000만원
  2. 1회 최대 손실: 5,000만 × 2% = 100만원
  3. 손절폭: -7% (매수가 대비)
  4. 최대 매수금액: 100만원 / 7% = 약 1,430만원
  5. 즉, 이 종목에 1,430만원 이상 넣으면 안 됨

19-3. 투자 심리 함정 5가지

FOMO (놓칠까 봐 두려움)

"나만 못 타고 있어!" → 추격매수 → 고점에 물림
해결: 기회는 항상 온다. 내 원칙에 맞을 때만 산다.

손실 회피 (Loss Aversion)

손절 못 하고 "다시 오르겠지" → 작은 손실이 큰 손실로
해결: 매수 시 손절선 미리 설정. 기계적 실행.

해결: 매수 후 일부러 반대 의견(악재) 찾아보기.

과신 (Overconfidence)
